실행 가이드 — 자사몰에 어떻게 적용하나

윤식단 자사몰은 아임웹(Imweb) 기반입니다. 실제 코드를 확인해, ① 데이터 통합(권장)과 ② 즉시 적용 코드 두 경로로 정리했습니다. 아래 코드는 yundiet.com의 실제 셀렉터를 타겟합니다.

먼저 · 문제의 핵심

0리뷰가 '0개'인 진짜 이유

디자인을 아무리 바꿔도, 보여줄 리뷰가 자사몰에 없으면 빈 화면입니다. 윤식단의 리뷰는 네이버 스마트스토어·쿠팡에 쌓여 있고 자사몰(아임웹)엔 통합돼 있지 않습니다. 그래서 순수단백 상품도 구매평 (0)으로 보입니다.

결론: ① 데이터 통합(타 채널 리뷰 → 자사몰)과 ② UI 개선(아래 코드)을 함께 해야 합니다. 통합은 CREMA가, UI 폴리시는 코드가 담당합니다.
두 가지 경로

1무엇을 선택하나

경로 A · CREMA 연동 권장

아임웹 공식 연동. 네이버·쿠팡 리뷰를 자사몰로 자동 수집하고, 분포·포토·필터·정렬 위젯을 제공. "리뷰 0개"를 근본 해결.

  • 설정 위주(코딩 최소)
  • 프로/글로벌 요금제 1개월 무료체험
  • 데이터 통합 = 가장 큰 임팩트

경로 B · 커스텀 코드 즉시

아임웹에 CSS/JS를 붙여 상단 별점 바·별점 분포·키워드·카드 배지를 즉시 노출. 집계 수치를 직접 입력해 바로 신뢰 지표를 보여줌.

  • CREMA 도입 전/후 모두 사용
  • 아래 전체 코드 제공
  • 상단 바·카드 배지는 CREMA와 병행 가능
추천 순서: 경로 B(코드)로 오늘 바로 상단 별점 바·카드 배지를 켜고 → 경로 A(CREMA)로 이번 달 안에 네이버/쿠팡 리뷰를 통합. 그러면 빈 화면 없이 단계적으로 좋아집니다.
경로 A

2CREMA 연동 (데이터 통합)

  1. 아임웹 앱/연동에서 'CREMA(크리마) 리뷰' 신청 — 아임웹은 2024년부터 크리마와 공식 연동. 프로/글로벌 요금제는 1개월 무료체험.
  2. 크리마 관리자에서 채널 리뷰 가져오기 설정 — 네이버 스마트스토어·쿠팡·지그재그 등 기존 리뷰를 자사몰로 import. (이 단계가 "0개" 문제를 해결)
  3. 아임웹 디자인 모드에서 위젯 배치 — 상품 상세에 '리뷰 위젯', 목록에 '별점 위젯', 상단에 '리뷰 요약'을 드래그로 삽입.
  4. 브랜드 톤 맞춤 — 아래 yundiet-review.css의 색·라운드 값으로 위젯을 윤식단 톤(올리브/크림)에 맞춤.
  5. 수집 자동화 — 구매 후 알림톡 리뷰 요청 + 포토/동영상 차등 적립으로 신규 리뷰 수집률을 끌어올림.
팁: CREMA를 켜면 분포/카드 위젯은 CREMA가 그립니다. 그땐 yundiet-review.js의 INJECT_WIDGET=false로 두고, 상단 별점 바 + 카드 배지만 우리 코드로 유지하면 충돌이 없습니다.
경로 B · 실제 코드

3아임웹에 붙여넣기

어디에 붙이나 (아임웹 위치)

  1. 관리자 [사이트 설정 > 고급]의 head/body 코드 입력으로 이동. (또는 [디자인 모드 > 코드 입력] 위젯을 상품 상세/목록 페이지에 추가)
  2. <head> 영역에 아래 CSS를 <style>…</style>로 감싸 붙여넣기.
  3. </body> 직전 영역에 아래 JS를 <script>…</script>로 감싸 붙여넣기.
  4. JS 상단 DATA에 상품별 집계 수치(네이버/쿠팡 합산 평균·리뷰수·분포·키워드)를 입력. 키는 shop_view?idx=NNN의 숫자.
  5. 상품 상세/목록 페이지를 새로고침해 상단 별점 바·분포·키워드·카드 배지 노출 확인.
가장 빠른 테스트(호스팅 참조): 아래 두 줄만 붙여도 동작합니다. 운영에는 인라인(직접 붙여넣기)을 권장합니다.
