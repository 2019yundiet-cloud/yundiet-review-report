2

리뷰를 "보여주기"

평균 별점·별점 분포·포토 갤러리·만족 키워드를 상품 상단과 목록 카드에 노출. 지금은 8.7만px 페이지 맨 끝에 텍스트로만 묻혀 있습니다.

3

리뷰로 "팔기"

다이어트 특화 리뷰(목적·재구매·비포애프터)와 베스트 후기를 홈·광고 소재로 재활용. 리뷰가 콘텐츠이자 전환 무기가 됩니다.

가장 빠른 한 방. 아임웹은 2024년부터 크리마(CREMA)와 공식 연동됩니다. 프로/글로벌 요금제는 1개월 무료 체험 가능 — 네이버/쿠팡 리뷰 import만 켜도 "구매평 0개" 화면이 수천 건으로 바뀝니다.
Part 1 · 벤치마크 해부

01라라스윗은 리뷰를 이렇게 다룬다

라라스윗은 Cafe24 + 알파리뷰(AlphaReview) 조합을 쓰고, 리뷰를 단순 "게시판"이 아니라 전환 엔진으로 운영합니다. 상품 상세 안에 리뷰 위젯을 깊게 통합하고, 리뷰 전용 메뉴를 글로벌 내비에 배치했습니다.

상품 상세 리뷰 위젯 — ★4.6 대문 별점 + 별점 분포 막대 + 포토&동영상 필터 + 정렬. 누적 6.7만 건 노출.
리뷰 전용 게시판(REVIEW) — 누적 11,992건. 추천순/최신순/평점 정렬, 상품 태그, 거의 모든 글에 실사용 포토 다수.

리뷰 위젯을 만드는 9가지 구성요소

★

평균 별점 대문

★4.6을 큼직하게 노출 + 누적 리뷰 수(6.7만)를 강조해 첫인상에서 신뢰를 박아넣음.

▦

별점 분포 막대

5★~1★ 각 비율을 막대로 시각화. 낮은 별점까지 투명하게 보여줘 오히려 신뢰도↑.

📷

포토·동영상 필터

시각 리뷰만 골라보기 토글. UGC(실사용 사진)를 전면에 배치.

⇅

3종 정렬

추천순(베스트)·최신순·평점 높은순. 베스트 리뷰가 기본으로 먼저 보임.

🧑

리치 리뷰 카드

아바타·닉네임·날짜·별점·본문 + 사진 2~4장. 정보 밀도가 높음.

🏷️

상품 태그

어떤 상품 리뷰인지 표기 → 게시판에서도 상품 탐색이 가능.

🧭

리뷰 전용 메뉴

글로벌 내비에 REVIEW 탭. 리뷰 자체가 랜딩/콘텐츠 역할.

👍

추천·답글

좋아요/답글로 상호작용 유도 → 리뷰가 커뮤니티처럼 살아있음.

🔁

전 상품 리뷰 풀링

위젯이 연관·전체 리뷰까지 끌어와 신상품도 "리뷰 0개"가 안 됨.

전략적 포인트. 라라스윗은 리뷰를 "고객이 찾아 들어가는 게시판"이 아니라 "구매 동선 위에 깔아두는 신뢰 레이어"로 씁니다. 별점은 상단에, 포토는 전면에, 베스트는 기본 정렬 — 스크롤하다 자연스럽게 후기를 밟고 지나가게 설계돼 있습니다.

Part 2 · 현재 진단

02윤식단 자사몰의 현재 리뷰

윤식단은 아임웹(Imweb) 기본 내장 리뷰를 그대로 사용 중입니다. 별도 리뷰 솔루션이 없고, 리뷰는 상세페이지 최하단에 단순 목록으로만 존재합니다.

순수단백 상품의 구매평 영역 — "구매평(0) · 등록된 구매평이 없습니다." 평균 별점·분포·요약 전부 없음.
상품 목록 카드 — 이름·가격·할인만 노출. 별점/리뷰 수 배지가 없어 목록 단계의 소셜프루프가 0.

진단: 6가지 구조적 빈틈

  • ✕
    리뷰가 자사몰에 안 모임. 네이버 스마트스토어·쿠팡엔 후기가 쌓이는데 자사몰은 채널별로 단절돼 "구매평 0개"로 보임. (가장 큰 손실)
  • ✕
    평균 별점·분포 차트가 없음. 첫인상에서 신뢰를 줄 요약 지표가 화면에 존재하지 않음.
  • ✕
    목록·상단에 소셜프루프 부재. 상품 카드와 가격/구매 버튼 근처에 별점이 없어, 사는 순간에 후기가 보이지 않음.
  • ✕
    리뷰가 페이지 맨 끝에 매몰. 8.7만px 길이 상세 최하단에 위치. 사실상 능동적으로 찾아야 보임.
  • ✕
    수집 동력이 약함. 적립금(일반 200/장문 +200/포토 +200)은 있으나, 구매 후 리마인드·차등 보상·간편 작성 흐름이 미흡.
  • △
    다이어트 맥락이 안 담김. 식단 카테고리의 핵심인 목적(체중·혈당·근성장)·재구매·비포애프터를 담을 양식이 없음. 기회
핵심 진단. 윤식단의 문제는 "리뷰가 적다"가 아니라 "있는 리뷰조차 안 보이고, 없는 자리(자사몰)엔 안 모은다" 입니다. 도구 1개 연동 + 노출 위치 재설계로 단기간에 체감이 바뀝니다.
